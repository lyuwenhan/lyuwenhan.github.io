# Creeper No Break (防爆苦力怕)

一个适用于 Fabric 的服务端模组，用于阻止苦力怕爆炸破坏方块。

苦力怕仍会爆炸、造成伤害并产生击退效果，但爆炸不会破坏地形或玩家建造的结构。

## 支持版本

- Minecraft 26.1.2
- Fabric Loader 0.19.2+
- Fabric API 0.150.0+
- Java 25
