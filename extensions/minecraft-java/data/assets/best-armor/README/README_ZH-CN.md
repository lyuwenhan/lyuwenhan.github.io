# Best Armor (最佳装备)

一个轻量级的 **客户端 Fabric 模组**，会向管理员创造模式物品栏添加三个预设装备潜影盒和一把虚空之剑。

## 概述

Best Armor 会向仅管理员可见的创造模式物品栏添加以下物品：

- 黄色潜影盒
- 橙色潜影盒
- 红色潜影盒
- 虚空之剑

潜影盒内的物品按照实际槽位顺序列出；每件物品的附魔、组件和属性均直接写在该物品下方。

### 虚空之剑

虚空之剑是独立添加到管理员创造模式物品栏中的满强度武器。

- 虚空伤害类型
- 使用钻石剑物品模型
- 强制显示附魔光效
- 不可损坏
- +2147483647 攻击伤害
- +2147483647 攻击速度
- 64 格实体攻击距离
- 360 度格挡角度
- 格挡延迟为 0
- 格挡时不消耗耐久
