fishing 游戏食用指南

食用方法 1: 本地运行

警告: 目前只兼容 Linux 系统 (编写环境: NOI Linux)。请勿在 Windows 中运行！
1. 前往 github 下载或直接下载 压缩包
2. 解压压缩包
3. 根据 using method.txt 的内容进行游玩

食用方法 2: 线上游玩

你可以使用 任意设备 进行游玩 (根据实测, 支持大部分主流浏览器和大部分设备) ,
现在存在如下几个官方可用网站
1. https://wh-fi.duckdns.org:8501/
2. https://lyuwenhan.github.io/fishing-website/
3. https://wh-fi.github.io/
4. https://wh-fi.pages.dev/
如上几个网站提供的内容几乎相同, 账号可以互通, 但不保证更新足够及时。

主要功能

钓鱼

使用 字符画 进行钓鱼画面渲染, 风格清新, 资源占用量 几乎为0
等待一定时间后, 就会有鱼出现。

小心, 鱼可能会脱钩。
