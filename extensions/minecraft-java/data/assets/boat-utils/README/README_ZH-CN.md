# Boat Utils (船只实用工具)

一个客户端 Fabric 模组，为船只提供可配置的视角与移动功能。

## 功能

- 解除原版乘船时的视角旋转限制
- 使船只跟随鼠标控制的视角方向
- 启用跟随视角后，将左右移动键从转向改为水平侧移
- 使船只使用蓝冰的摩擦系数
- 在水下以每秒 2.4 格的固定速度上浮，防止船只下沉
- 开启防止下沉后，允许水下上船并阻止原版强制下船
- 提供可配置的船只步高

### 解除视角限制

移除原版乘船时的 180 度水平视角限制，使视角可以完整旋转一周。

### 跟随视角

使船只立即朝向鼠标控制的视角方向。

启用后：

- 前进和后退会相对于当前视角方向移动
- 左右移动键改为水平侧移，不再旋转船只

关闭后，船只恢复原版转向操作。

### 速度提升修复

使船只在使用蓝冰的摩擦力

### 防止下沉

船只位于水下时，会给予一个垂直速度

开启此选项后，模组还会：

- 阻止原版水下强制下船计时将乘客弹出
- 允许玩家在水下登上船只

这些骑乘行为由服务端控制。单人游戏中预计可以生效，但远程服务器可能拒绝或纠正这些行为。

### 船只踏步

设置船只移动时可直接跨越的最大高度为 0-10 格。

## 多人服务器风险

本模组会修改客户端主导的船只移动。服务器可能纠正、拒绝或标记这些移动行为。

仅应在服务器规则允许时使用。模组作者不对使用本模组造成的警告、踢出、封禁或其他处罚负责。
