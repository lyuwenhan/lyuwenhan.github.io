# Best Armor

一个轻量级 **客户端 Fabric 模组**，用于向管理员创造标签页添加三个预设潜影盒。

---

## 概述

Best Armor 会向管理员专用创造标签页添加三个预设潜影盒。

三个潜影盒分别对应三种装备定位：

- 黄色潜影盒：生存模式中能正常获得的最好装备。
- 橙色潜影盒：外表看起来和黄色装备几乎没有区别，但内部属性很高。
- 红色潜影盒：整体属性都很高，用作完整的高强度管理员装备预设。

每套装备都包含：

- 普通下界合金胸甲
- 原版鞘翅
- 带有 Glideplate 风格数据和滑翔支持的下界合金胸甲

本模组不会注册新的物品、方块、命令、界面或配置文件，只会直接在 Java 代码中生成已有物品栈，并把它们加入管理员创造标签页。

---

## 主要功能

- 向管理员创造标签页添加三个预设潜影盒
- 包含黄色、橙色、红色三套装备预设
- 使用带有预配置容器内容的原版潜影盒
- 每套都包含普通胸甲、原版鞘翅和 Glideplate 风格胸甲
- 所有物品都由 Java 代码直接生成
- 不从 txt 文件动态导入物品定义
- 不解析 `/give` 命令字符串
- 不注册自定义物品或方块
- 无需命令或配置
- 仅客户端运行
- 运行逻辑轻量

---

## 装备分级

### 黄色潜影盒

黄色潜影盒包含生存模式中能正常获得的最好装备。

它的定位是干净的顶级生存装备组，使用强力但接近原版生存逻辑的附魔和属性，不包含隐藏的极端属性。

### 橙色潜影盒

橙色潜影盒的外表设计成和黄色装备几乎没有区别。

它的装备在外观上保持普通顶级装备的样子，但内部属性明显更高，适合管理员使用，同时不会在视觉上显得过于异常。

### 红色潜影盒

红色潜影盒是最高强度预设。

其中装备拥有很高的附魔等级和很高的内部属性，用作完整的管理员高强度装备组。

---

## 胸甲与鞘翅配置

每个潜影盒都同时包含普通护甲和飞行相关装备。

普通下界合金胸甲会保留为普通胸甲，不带 Glideplate 物品数据，用作干净的常规护甲选项。

每个潜影盒也会额外包含一个原版鞘翅。

此外，每个潜影盒还会包含原来的 Glideplate 风格下界合金胸甲。这个胸甲带有 Glideplate 兼容物品数据和滑翔支持，可以作为带有鞘翅功能的胸甲使用。

Glideplate 风格胸甲带有与 [Glideplate](https://modrinth.com/mod/glideplate) 兼容的数据。

这些预设物品栈由本模组直接生成，不需要服务器安装 Glideplate。客户端安装 Glideplate 可以改善 Glideplate 相关的物品模型和装备渲染效果。

## 多人游戏说明

- 仅客户端运行
- 本模组不要求服务器安装
- 服务器仍然拥有最终判定权
- 创造物品栏中的修改物品栈可能会被服务器校验或拒绝
- 部分服务器不允许使用修改过的创造物品栏物品
- 仅应在允许此类管理员工具的多人服务器中使用
