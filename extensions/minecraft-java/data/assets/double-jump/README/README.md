# Double Jump

A lightweight client-side Fabric mod that adds configurable multiple jumping.

## Features

- Enable or disable the mod
- Set the total jump count from 2 to 10 with a slider
- Enter a custom jump count above 10
- Enable unlimited airborne jumps
- Configure through a vanilla-style settings screen
- Optional Mod Menu integration

## Usage

Press the normal jump key again while airborne to perform another jump. Landing resets the available jump count.

Because this mod is client-side only, remote servers may correct or reject modified movement.

## Supported Versions

- Minecraft 26.1.2
- Fabric Loader 0.19.2+
- Fabric API 0.150.0+
- Java 25
