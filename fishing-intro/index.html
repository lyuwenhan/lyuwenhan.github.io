fishing 游戏食用指南

食用方法 1: 本地运行

警告: 目前只兼容 Linux 系统 (编写环境: NOI Linux)。请勿在 Windows 中运行！
1. 前往 github 下载或直接下载 压缩包
2. 解压压缩包
3. 根据 using method.txt 的内容进行游玩

食用方法 2: 线上游玩

你可以使用 任意设备 进行游玩,
具体连接请前往 这里 查看

钓鱼
