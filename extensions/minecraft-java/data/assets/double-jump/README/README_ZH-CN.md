# Double Jump (多段跳)

一个轻量的纯客户端 Fabric 模组，提供可配置的多段跳功能。

## 功能

- 启用或禁用模组
- 使用滑块设置 2 到 10 次总跳跃数量
- 使用输入框设置超过 10 的自定义跳跃数量
- 启用空中无限跳跃
- 使用原版风格的设置界面
- 可选的 Mod Menu 集成

## 使用方法

玩家在空中再次按下原版跳跃键即可继续跳跃。落地后会重置可用跳跃次数。

由于这是纯客户端模组，远程服务器可能修正或拒绝客户端修改后的移动。

## 支持版本

- Minecraft 26.1.2
- Fabric Loader 0.19.2+
- Fabric API 0.150.0+
- Java 25
