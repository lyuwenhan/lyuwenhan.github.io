# Best Armor (最佳装备)

一个轻量级的**客户端 Fabric 模组**，会在管理员创造模式物品栏中加入三个预设管理员潜影盒和一把虚空之剑。

## 概述

Best Armor 会在仅管理员可见的创造模式物品栏中加入四个动态生成的物品：

- 黄色潜影盒
- 橙色潜影盒
- 红色潜影盒
- 虚空之剑

三个潜影盒分别提供不同级别的装备预设：

- 黄色潜影盒包含一套以正常生存模式可获得附魔等级为主的高端装备。
- 橙色潜影盒在外观上仍接近原版装备，但加入了正常情况下互斥的附魔、极高防护等级、不可损坏组件、特殊靴子以及可阻挡所有已注册伤害类型的盾牌。
- 红色潜影盒包含完整强度的管理员装备，具有极高附魔、自定义战斗属性、扩展攻击距离、免疫摔落伤害的靴子以及其他无法通过正常生存模式获得的物品组件。

## 主要功能

- 添加了三个预设装备潜影盒
- 添加了虚空之剑
- 包含武器、工具、盔甲、鞘翅装备、盾牌和钓鱼竿

## 虚空之剑

虚空之剑会作为一个独立物品直接添加到管理员创造模式物品栏。

它在内部基于下界合金剑，但使用钻石剑的物品模型。

### 显示效果

- 名称：`Void Sword`
- 名称颜色：淡紫色
- 名称样式：粗体
- 物品模型：钻石剑
- 附魔光效：始终启用
- 不可损坏

### 伤害行为

该剑使用 `fell_out_of_world` 伤害类型。

这与虚空伤害使用的伤害类型相同，而不是普通近战伤害类型。

### 属性

无论持于主手还是副手，该剑都会提供：

- 攻击伤害：+2,147,483,647
- 攻击速度：+2,147,483,647

### 距离

- 最大配置攻击距离：64 格
- 最大配置创造模式攻击距离：64 格

### 格挡

该剑还会获得与管理员盾牌相同的全伤害格挡组件：

- 阻挡所有已注册伤害类型
- 使用 360 度格挡角度
- 格挡延迟为 0
- 格挡时不消耗耐久
- 使用极高伤害减免值

实际服务器行为仍然受服务器端物品验证和战斗权限控制。

## Glideplate 兼容性

每个潜影盒中都包含一件经过特殊配置、具有鞘翅功能的下界合金胸甲。

该胸甲会获得：

- `glideplate_has_elytra` 自定义数据
- `gliding` 自定义数据
- `gliding_chestplate_has_elytra` 自定义数据
- Glideplate 兼容自定义模型数据
- 原版滑翔组件
- 翻译名称 `Netherite Chestplate with Elytra`
- 描述文本 `With Elytra`

这些组件使该物品无需注册新的物品类型，即可作为可滑翔胸甲使用。

生成的胸甲使用与 [Glideplate](https://modrinth.com/mod/glideplate) 兼容的数据。

安装 Glideplate 可能会改善相关物品名称、模型和视觉表现，但 Best Armor 会直接生成这些物品堆栈。

## 创造模式物品栏行为

只有在管理员限制物品可见时，生成的物品才会被添加到管理员创造模式物品栏。

以下物品会同时出现在管理员物品栏主页面和创造模式搜索结果中：

- 黄色潜影盒
- 橙色潜影盒
- 红色潜影盒
- 虚空之剑

这些物品只会在客户端世界已加载时生成，因为本模组需要使用当前注册表访问来解析附魔和伤害类型。

## 技术行为

Best Armor 会直接创建 `ItemStack` 实例，并为其设置原版数据组件。

本模组使用的组件包括：

- 附魔
- 不可损坏
- 属性修饰符
- 容器内容
- 自定义数据
- 自定义模型数据
- 滑翔组件
- 物品名称
- 物品描述
- 攻击距离
- 攻击格挡
- 伤害类型
- 附魔光效覆盖
- 物品模型

本模组不会：

- 注册自定义物品类型
- 注册自定义方块
- 注册命令
- 添加配置文件
- 添加界面
- 从 JSON 或文本文件读取物品定义
- 执行 `/give` 命令
- 解析命令字符串

## 视觉行为

- 黄色、橙色和红色潜影盒都显示为原版对应颜色的潜影盒
- 每个潜影盒都使用自定义物品名称
- 三个潜影盒都会显示在管理员创造模式物品栏和搜索结果中
- 虚空之剑使用钻石剑模型
- 虚空之剑始终显示附魔光效
- Glideplate 风格胸甲包含自定义名称、描述和模型数据
- 本模组不包含自定义纹理
- 本模组不添加自定义用户界面或 HUD

## 多人游戏安全性

- 仅客户端运行
- 模组本身不要求服务器安装
- 服务器仍然拥有物品栏和战斗行为的最终控制权
- 服务器可能会验证、修改、拒绝或移除这些生成的物品堆栈
- 不受支持的附魔组合在不同服务器上的行为可能不一致
- 自定义属性和数据组件可能会被服务器端验证拒绝
- 虚空之剑和管理员装备可能会严重破坏正常游戏平衡
- 仅应在私人世界或明确允许此类管理员装备的服务器中使用
