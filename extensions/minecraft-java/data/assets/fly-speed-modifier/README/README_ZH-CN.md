# Fly Speed Modifier (飞行速度修改器)

一个客户端 Fabric 模组，可在按住调速键并滚动鼠标滚轮时，临时调整 Freecam 和玩家飞行速度。

## 功能

- 修改原版创造模式飞行速度
- 安装并启用 [Freecam](https://modrinth.com/mod/freecam) 时，可修改 Freecam 移动速度
- 可选择将倍数应用于其他方式移动的速度
- 按住调速键并滚动鼠标滚轮即可调整倍数
- 支持配置最小值、最大值、默认值和滚轮步长
- 支持选择每次开始调整时是否重置倍数
- 关闭全范围取值时，最大倍数为 `20倍`；开启时，最大倍数为 `100倍`
- 鼠标滚轮使用非线性方式调整倍数
- 默认调速键为左 Alt

## 使用方法

1. 进入受支持的移动状态，例如创造飞行、Freecam，或在启用“应用于其他移动”后进入对应移动状态。
2. 按住左 Alt。
3. 滚动鼠标滚轮调整倍数。
4. 当前值会显示为 `速度倍数：2.00倍`。

## 配置

### 全范围取值

- 默认关闭
- 关闭：最大可选倍数为 `20倍`
- 开启：最大可选倍数为 `100倍`
- 修改此选项时，速度最大值和速度默认值滑块会立即更新范围

### 速度最小值

设置临时速度倍数的最低值。

设置为 `0倍` 时，受支持路径中新产生的移动会被禁用；但已有惯性、击退或特殊冲量仍可能按照原版逻辑继续，除非对应移动路径本身也被倍率处理。

### 速度最大值

设置临时速度倍数的最高值；其可选范围由“全范围取值”决定。

### 速度默认值

设置首次调整时使用的倍数，以及启用“调整前重置”后每次开始调整时使用的倍数。

默认值的可选范围始终限制在当前最小值和最大值之间。

### 调整前重置

- 启用：按下调速键时先重置为速度默认值
- 禁用：按下调速键时保留上一次倍数

### 滚轮倍数步长

控制每一个鼠标滚轮单位对应的倍数调整速度。

### 应用于其他移动

**应用于其他移动**选项默认关闭。

启用后，当前倍数还会应用于以下受支持的移动方式：

- 行走、疾跑、潜行和普通水平移动
- 未飞行时的空中水平控制
- 游泳和岩浆中的主动移动
- 水中的主动上浮和下潜
- 梯子、藤蔓及其他可攀爬方块上的移动
- 使用标准骑乘移动路径、由本地玩家控制的生物坐骑，包括马、猪、炽足兽、骆驼和快乐恶魂
- 船和竹筏的推进
- 鞘翅 X、Y、Z 三个方向的实际移动，包括俯冲、爬升和自然下沉

该选项不会主动修改：

- 玩家的普通跳跃初速度
- 马匹跳跃强度
- 击退、爆炸或碰撞冲量
- 船的转向速度
- 矿车移动
- 在粉雪中跳跃

## 多人服务器风险

这是客户端移动修改模组。远程服务器可能纠正、拒绝或标记异常移动。高倍率可能造成位置不同步、触发反作弊、断开连接或封禁。仅应在服务器规则明确允许时使用。模组作者不对使用本模组造成的处罚负责。

鞘翅使用高倍率时，玩家可能在一个 Tick 内跨越非常大的距离。使用烟花、接近未加载区块、地形或世界边界时，应使用较低倍数。

## 支持版本

- Minecraft 26.1.2
- Fabric Loader 0.19.2+
- Fabric API 0.150.0+
- Java 25
- Mod Menu（可选）
