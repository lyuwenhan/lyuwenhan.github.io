# Fly Speed Modifier (飞行速度修改器)

一个客户端 Fabric 模组。按住按键并滚动鼠标滚轮时，可以临时调整 Freecam 和直接飞行的速度倍数。

## 功能

* 支持 Freecam 速度倍数调整
* 支持普通 / 直接玩家飞行速度倍数调整
* 按下调速键时，将倍数重置为 `1 倍`
* 安装 Cloth Config 后，可以通过 Mod Menu 打开配置界面
* 支持配置最小倍数、最大倍数和滚轮调整速度
* 按住 Alt 键调整飞行速度倍数
* 最大倍数滑块使用非线性映射，方便在低倍数范围内精细调整
* 鼠标滚轮使用非线性方式调整倍数
* 支持原版创造模式飞行，以及 [Freecam](https://modrinth.com/mod/freecam)

## 支持版本

* Minecraft 26.1.2
* Fabric Loader 0.19.2+
* Fabric API 0.150.0+
* Java 25
* Freecam（可选）
* Cloth Config 26.1.154+fabric（可选）
* Mod Menu 18.0.0-beta.1（可选）
