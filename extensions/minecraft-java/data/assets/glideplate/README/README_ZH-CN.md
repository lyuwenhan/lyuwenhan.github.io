# Glideplate (附翼胸甲)

附翼胸甲是一个 Fabric 模组，允许玩家在铁砧中将胸甲与鞘翅合成为可滑翔胸甲，并为合成后的物品提供对应的客户端视觉效果。

将胸甲放入铁砧左侧槽位，将鞘翅放入右侧槽位。输出物会保留胸甲本身的护甲属性，同时获得滑翔支持，并携带用于识别附翼胸甲的物品数据。

当客户端安装本模组时，被标记为附翼胸甲的物品会获得更清晰的显示效果：物品模型会在胸甲下方显示鞘翅，玩家装备后也会渲染鞘翅翅膀。

## 功能

- 支持皮革、铜、锁链、铁、金、钻石和下界合金胸甲
- 仅支持一种合成方向：胸甲放左侧，鞘翅放右侧
- 保留胸甲的耐久、附魔、盔甲纹饰和其他物品数据
- 忽略鞘翅的耐久、损坏值和附魔
- 添加 Minecraft 的 glider 组件，使物品可以用于鞘翅滑翔
- 添加自定义物品数据和自定义模型数据，用于识别附翼胸甲
- 为被标记的胸甲更新物品模型，显示胸甲与鞘翅结合后的外观
- 普通胸甲仍然保持原本的渲染效果
- 装备附翼胸甲时显示鞘翅翅膀
- 添加本地化物品名称和提示文本，并提供英文回退文本
- 合成不消耗经验
- 不增加铁砧的累积惩罚
- 支持专用服务器

## 使用方法

使用铁砧合成：

1. 将支持的胸甲放入左侧槽位。
2. 将鞘翅放入右侧槽位。
3. 取出输出的附翼胸甲。

输出物仍然是一件胸甲，但同时可以用于鞘翅滑翔。

## 兼容性

附翼胸甲可以安装在服务器、客户端，或两者同时安装。

安装在服务器上时，会启用铁砧合成和实际玩法逻辑。玩家无需安装本模组也可以加入服务器，但如果客户端未安装本模组，附翼胸甲会像普通胸甲一样显示。

安装在客户端上时，可以显示增强后的附翼胸甲物品模型和装备后的鞘翅翅膀。

## 支持版本

- Minecraft 26.1.2
- Fabric Loader 0.19.2+
- Fabric API 0.150.0+
- Java 25
