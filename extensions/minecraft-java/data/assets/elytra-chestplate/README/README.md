# Elytra Chestplate

A Fabric mod that lets players combine a chestplate with an elytra in an anvil.

The result is a new chestplate item that keeps the armor properties of the original chestplate and can also be used for elytra gliding.

## Features

- Supports leather, copper, chainmail, iron, golden, diamond and netherite chestplates
- Combines a chestplate and an elytra in an anvil
- Keeps the chestplate's trims, durability, enchantments and other item data
- Ignores the elytra's durability, damage and enchantments
- Costs 0 XP
- Does not increase anvil prior-work penalty
- Can be worn as a chestplate
- Can be used for elytra gliding

## Supported Versions

- Minecraft 1.21.11
- Fabric Loader 0.18+
- Fabric API
- Java 21
