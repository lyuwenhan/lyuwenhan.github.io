# Glideplate

Glideplate is a client-side visual companion for Glideplate items.

It recognizes marked chestplates and gives them clearer visuals in the client: the item model shows an elytra under the chestplate, and equipped Glideplates render with elytra wings.

## Features

- Updates marked chestplate item models with an elytra-and-chestplate appearance
- Supports leather, copper, chainmail, iron, golden, diamond and netherite chestplates
- Keeps normal chestplates rendering normally
- Shows elytra wings when a marked chestplate is equipped
- Adds English and Simplified Chinese language entries for Glideplate item names and tooltip text
- Does not add gameplay logic, recipes or server-side requirements

## Compatibility

Glideplate is client-only. It is useful when playing on a world or server that provides Glideplate-marked chestplates.

If this mod is not installed on a client, those items still exist and work normally, but they render like regular chestplates.

## Supported Versions

- Minecraft 1.21.11
- Fabric Loader 0.18+
- Fabric API
- Java 21
