# Glideplate (附翼胸甲)

一个 Fabric 模组，可以让玩家在铁砧中将胸甲与鞘翅进行合并。

合成后的物品仍然保留原版胸甲的护甲属性，同时也可以像鞘翅一样进行滑翔飞行。

## 功能

- 支持皮革、铜、锁链、铁、金、钻石和下界合金胸甲
- 可以在铁砧中将胸甲与鞘翅进行合成
- 保留胸甲的盔甲纹饰、耐久、附魔以及其它物品数据
- 忽略鞘翅的耐久、损坏值和附魔
- 合成不消耗经验
- 不增加铁砧历史惩罚
- 可以作为普通胸甲穿戴
- 可以像鞘翅一样进行滑翔飞行

## 支持版本

- Minecraft 1.21.11
- Fabric Loader 0.18+
- Fabric API
- Java 21
