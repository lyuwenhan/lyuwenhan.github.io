# Fly Speed Modifier

A client-side Fabric mod that temporarily multiplies Freecam and direct flight speed while you hold a key and scroll the mouse wheel.

## Features

* Supports Freecam speed multiplier adjustment
* Supports normal / direct player flight speed multiplier adjustment
* Resets the multiplier to `1x` when the adjust key is pressed
* Supports a Mod Menu configuration screen when Cloth Config is installed
* Supports configurable minimum multiplier, maximum multiplier, and scroll adjustment speed
* Hold Alt key to adjust flight speed multiplier
* Uses a non-linear max-multiplier slider for finer low-range control
* Uses non-linear mouse-wheel multiplier adjustment
* Supports Vanilla Creative Flight, and [Freecam](https://modrinth.com/mod/freecam)

## Supported Versions

* Minecraft 26.1.2
* Fabric Loader 0.19.2+
* Fabric API 0.150.0+
* Java 25
* Freecam (Optional)
* Cloth Config 26.1.154+fabric (Optional)
* Mod Menu 18.0.0-beta.1 (Optional)
