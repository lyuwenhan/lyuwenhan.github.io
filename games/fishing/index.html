The Fishing Game

How to use:

  1. Run Locally

    A Linux system is required. WSL is also supported.

    To run this project, follow the steps below:

    1. Download from Github or download the ZIP archive directly.
    2. Extract the ZIP archive.
    3. Follow the instructions in using-method.txt to start playing.

  2. Play online

    The game can be accessed using any of the links below.

    1. https://wh-fi.duckdns.org:8501/
    2. https://lyuwenhan.github.io/fishing-game-website/
    3. https://wh-fi.github.io/
    4. https://wh-fi.pages.dev/

Main Features

Fishing

The fishing scene is rendered using ASCII art.

After waiting for a certain amount of time, a fish will appear.

Be careful: the fish may escape.
