# Best Armor

A lightweight **client-side Fabric mod** that adds three prebuilt equipment shulker boxes and a Void Sword to the operator creative tab.

## Overview

Best Armor adds the following dynamically generated items to the operator-only creative tab:

- Yellow Shulker Box
- Orange Shulker Box
- Red Shulker Box
- Void Sword

Items inside each shulker box are listed in their actual slot order. Every enchantment, component, and attribute is written directly under the relevant item.

### Void Sword

The Void Sword is a standalone full-power weapon added to the operator creative tab.

- Void damage type
- Uses the diamond sword item model
- Forced enchantment glint
- Unbreakable
- +2147483647 Attack Damage
- +2147483647 Attack Speed
- 64-block entity attack range
- 360-degree blocking angle
- Zero blocking delay
- No durability loss while blocking
